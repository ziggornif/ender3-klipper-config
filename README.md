# Ender 3 Klipper Configuration

Klipper configuration for modified Ender 3.

## Hardware

- **Motherboard**: Creality v4.2.7
- **Controller**: Raspberry Pi 4
- **Extruder**: Creality Sprite Extruder Pro
- **X Axis**: Linear rail
- **Y Axis**: Linear rail
- **Z Axis**: Dual Z
- **Auto-leveling**: BLTouch
- **Input Shaping**: Raspberry Pi Zero + ADXL345 accelerometers

## Features

- Auto bed leveling with BLTouch
- 7x7 bed mesh
- Input shaper calibrated (MZV)
- PID tuned (extruder and heated bed)
- Screws tilt adjust for mechanical leveling
